## Redis（Remote dictionary server）
基于内存的数据存储系统

### 使用方式
- CLI（Command Line Interface）：通过Redis-CLI命令行工具来使用Redis
- API（Application Programming Interface）：通过常用编程语言使用编写代码的方式来使用Redis
- GUI（Graphical User Interface）：通过图形化工具来使用Redis

### 优势
- 性能极高
- 数据类型丰富，单键值对最大支持512M大小的数据
- 简单易用，支持所有主流编程语言
- 支持数据持久化、主从复制、哨兵模式等高可用特性

### 启动
- 获取Redis版本信息以及端口号等信息：`redis-server`
- 启动Redis命令行工具：`redis-cli`
- 以原始形式显式内容：`redis-cli --raw`
- 关闭Redis：`quit`
