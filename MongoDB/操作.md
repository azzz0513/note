### 查询
#### 比较运算符
| 运算符  | 作用     |        |
| ---- | ------ | ------ |
| $gt  | 大于     | >      |
| $gte | 大于等于   | >=     |
| $lt  | 小于     | <      |
| $lte | 小于等于   | <=     |
| $ne  | 不等于    | !=     |
| $in  | in     | in     |
| $nin | not in | not in |
#### 聚合查询
常用通道

| 符号     | 作用               |
| ------ | ---------------- |
| $group | 将集合中的文档分组，用于统计结果 |
| $match | 过滤数据，只输出符合条件的文档  |
| $sort  | 聚合数据进一步排序        |
| $skip  | 跳过指定文档数          |
| $limit | 限制集合数据返回文档数      |
常用表达式

| 符号   | 作用                        |
| ---- | ------------------------- |
| $sum | 总和（`$sum: 1`可表示统计分组中文档数量） |
| $avg | 平均                        |
| $min | 最小值                       |
| $max | 最大值                       |

### 文档
#### 插入文档
```Bash
# 插入一个数据
db.users.insertOne({
    name: "John Doe",
    email: "john@example.com",
    age: 30,
    created_at: new Date()
})
 
# 插入多个数据，可以插入不同结构数据
db.users.insertMany([
    { name: "Jane Doe", email: "jane@example.com", age: 25 },
    { name: "Alice Smith", email: "alice@example.com", addr: "GuangZhou" }
])
```


#### 查询文档
```Bash
db.users.find()  // 查询所有文档
db.users.find({ name: "John Doe" })  // 查询特定文档
db.users.find({ age: { $gt: 25 } })  // 查询年龄大于25的文档
db.users.find().pretty()  // 格式化输出查询结果
db.users.find({, {name:1, _id: 0}) // 只显示name列
db.users.find().sort({age:-1}) // 1 升序，-1 降序

// 分页
db.users.find().limit(2) // 查询前两条数据
db.users.find().skip(2).limit(2) // 跳过两条，查询两条数据
```

#### 更新文档
```Bash
db.users.updateOne({ name: "John Doe" }, { $set: { age: 31 } })
db.users.updateMany({ age: { $lt: 30 } }, { $set: { isYoung: true } })
```

#### 删除文档
```Bash
db.users.deleteOne({ name: "John Doe" })
db.users.deleteMany({ age: { $lt: 30 } })
```

### 索引
索引是MongoDB中用于加速查询的重要工具。合理使用索引可以显著提高查询性能
#### 创建索引
```Bash
db.users.createIndex({ name: 1 })  // 升序索引
db.users.createIndex({ age: -1 })  // 降序索引
```

#### 删除索引
```Bash
db.users.dropIndex({ name: 1 })
```

### 用户管理
#### 创建用户
```Bash
use admin
db.createUser({
    user: "azzz",
    pwd: "123456",
    roles: [{ role: "readWrite", db: "my_im_system" }]
})
```
#### 授权
```Bash
use mydatabase
db.grantRolesToUser("newuser", [{ role: "readWrite", db: "mydatabase" }])
```
#### 删除用户
```Bash
db.dropUser("newuser")
```
#### 更改用户密码
```Bash
db.changeUserPassword("newuser", "newpassword")
```

### 备份与恢复
#### 备份数据库
```Bash
mongodump --db mydatabase --out /path/to/backup
```

#### 恢复数据库
```Bash
mongorestore --db mydatabase /path/to/backup/mydatabase
```
