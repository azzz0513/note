### docker安装MongoDB
拉取镜像
```Bash
docker pull bitnami/mongodb:latest
```
创建容器：
```Bash
docker run --name mongodb -p 27017:27017 -v C:\mongo:/data/db -d bitnami/mongodb:latest
```
创建用户：
```Bash
db.createUser({
	user: "root1", 
	pwd: "123456", 
	roles: [{ role: "dbOwner", db: "test" }]
})
# 验证用户是否创建成功
show users
# 后续进入数据库
mongosh -u root1 -p 123456 test
```
