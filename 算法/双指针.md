## 相向双指针
两个指针`left = 0`，`right = n - 1`，从数组的两端开始，向中间移动，这叫相向双指针。**滑动窗口**相当于同向双指针
例：
![[images/Pasted image 20250728112332.png]]
代码：
```Go
func reverseString(s []byte) {
    left, right := 0, len(s)-1
    for left < right {
        s[left], s[right] = s[right], s[left]
        left++
        right--
    }
    return
}
```

## 同向双指针
两个指针的移动方向相同（都向右，或者都向左）

例：
![[images/Pasted image 20250823223624.png]]
代码：
```Go
func triangleNumber(nums []int) int {
    sort.Ints(nums)
    n := len(nums)
    if n < 3 {
        return 0
    }
    ans := 0
    for i, x := range nums[:n-2] {
        if x == 0 {
            continue
        }
        j := i + 1
        for k := i + 2; k < n; k++ {
            for nums[k]-nums[j] >= x {
                j++
            }
            ans += k - j
        }
    }
    return ans
}
```

```Go
func findLengthOfShortestSubarray(arr []int) int {
    n := len(arr)
    right := n - 1
    for right > 0 && arr[right-1] <= arr[right] {
        right--
    }
    if right == 0 {
        return 0
    }
    ans := right
    for left := 0; left == 0 || arr[left-1] <= arr[left]; left++ {
        for right < n && arr[right] < arr[left] {
            right++
        }
        ans = min(ans, right-left-1)
    }
    return ans
}
```
