## 枚举右，维护左
对于双变量问题，例如两数之和`a+b=t`，可以枚举右边的a，转换成单变量问题，也就是在b左边查找是否又a=t-b，这可以用哈希表维护。

例：
![[images/Pasted image 20250815151557.png]]
代码：
```Go
func minimumCardPickup(cards []int) int {
    cnt := map[int]int{}
    ans := math.MaxInt
    for i, card := range cards {
        if j, ok := cnt[card]; ok {
            ans = min(ans, i-j+1)
        }
        cnt[card] = i
    }
    if ans == math.MaxInt {
        return -1
    }
    return ans
}
```

## 枚举中间
对于三个或者四个变量的问题，枚举中间的变量往往更好算。
为什么？比如问题有三个下标，需要满足 0≤i<j<k<n，对比一下：
- 枚举 i，后续计算中还需保证 j<k。
- 枚举 j，那么 i 和 k 自动被 j 隔开，互相独立，后续计算中无需关心 i 和 k 的位置关系。
例：
![[images/Pasted image 20250823223419.png]]
代码：
```Go
func minimumSum(nums []int) int {
    ans := math.MaxInt
    n := len(nums)
    preMin, sufMin := make([]int, n), make([]int, n)
    preMin[0], sufMin[n-1] = nums[0], nums[n-1]
    for i := 1; i < n-1; i++ {
        preMin[i] = min(preMin[i-1], nums[i-1])
        sufMin[n-i-1] = min(sufMin[n-i], nums[n-i])
    }
    for i := 1; i < n-1; i++ {
        num := nums[i]
        if num > preMin[i] && num > sufMin[i] {
            ans = min(ans, num+preMin[i]+sufMin[i])
        }
    }
    if ans == math.MaxInt {
        return -1
    }
    return ans
}
```
