# 常用的排序算法
#### 使用`slices`包
- `slices.Sort(x)`：对切片进行升序排序（支持内置类型如int、string、float64）
- `slices.SortFunc(x, cmp)`：自定义排序规则。注意cmp函数返回的是int（负数表示小于，0表示相等，正数表示大于），而不是bool
```Go
import (
    "fmt"
    "slices"
)

func main() {
    nums := []int{5, 2, 9, 1, 5}
    
    // 使用 pdqsort
    slices.Sort(nums) 
    fmt.Println(nums) // [1 2 5 5 9]

    // 自定义比较逻辑
    slices.SortFunc(nums, func(a, b int) int {
        return b - a // 降序
    })
}
```

#### 使用`sort`包
- 针对基本类型的快捷排序：
	- `sort.Ints(x)`
	- `sort.Strings(x)`
	- `sort.Float64s(x)`
- 自定义排序：
	- `sort.Slice(x, less func(i, j int) bool)`：你需要提供一个函数，告诉它下标 i 的元素是否**小于**下标 j 的元素。
```Go
type User struct { Name string; Age int }
users := []User{{"Bob", 20}, {"Alice", 25}}

// 按 Age 升序
sort.Slice(users, func(i, j int) bool {
    return users[i].Age < users[j].Age
})

// 按 Age 降序
sort.Slice(users, func(i, j int) bool {
    return users[i].Age > users[j].Age
})
```

### `slices`
#### 排序和查找
- **slices.Sort(x)**
	- **用途**：对基本类型（int, float, string）切片进行升序排序。
	- **底层**：pdqsort。
```Go
nums := []int{4, 2, 8}
slices.Sort(nums) // [2 4 8]
```

- **slices.SortFunc(x, cmp)**
	- **用途**：自定义排序规则（常用于结构体）。
	- **示例**：按年龄降序排序。
```Go
type User struct { Name string; Age int }
users := []User{{"A", 20}, {"B", 30}}

slices.SortFunc(users, func(a, b User) int {
    return b.Age - a.Age // 返回正数、负数或0
})
```

- **slices.Reverse(x)**
	- **用途**：原地反转切片。
```Go
slices.Reverse(nums)
```

- **slices.BinarySearch(x, target)**
    - **用途**：在**已排序**的切片中查找元素。
	- **返回**：(index, found)。如果找到，found 为 true；如果没找到，index 是该元素应该插入的位置。

- **slices.Contains(x, target)**
	- **用途**：判断切片是否包含某元素（以前我们需要写循环遍历，现在直接用这个）。
```Go
has := slices.Contains([]string{"a", "b"}, "a") // true
```

- **slices.Max(x) / slices.Min(x)**
    - **用途**：直接获取切片中的最大值或最小值（切片不能为空，否则 panic）。

#### 修改切片
- **slices.Delete(x, i, j)**
	- **用途**：删除索引 i 到 j-1 的元素。
	- **注意**：它不修改底层数组长度，而是返回一个新的切片长度，**必须重新赋值**。
```Go
s := []int{0, 1, 2, 3, 4}
s = slices.Delete(s, 1, 3) // 删除索引1和2 (即数字1, 2)
// s 变为 [0 3 4]
```

- **slices.Insert(x, i, v...)**
	- **用途**：在索引 i 处插入一个或多个元素。
```Go
s := []int{1, 3}
s = slices.Insert(s, 1, 2) // 在索引1插入2 -> [1 2 3]
```

- **slices.Compact(x)**
	- **用途**：移除**连续重复**的元素（通常先排序再使用，相当于去重）。
```Go
s := []int{1, 1, 2, 3, 3, 3}
s = slices.Compact(s) // [1 2 3]
```

#### 比较与工具
- **slices.Equal(x, y)**
    - **用途**：判断两个切片内容是否完全相等（长度相同且元素相同）。
    - **痛点解决**：以前切片不能直接用 == 比较，必须写循环或用 reflect.DeepEqual（慢），现在用这个飞快。
- **slices.Clone(x)**
    - **用途**：浅拷贝一个切片。当你不想修改原切片时非常有用。

## 字符串查找和判断
- `Contains(s, substr string) bool`：判断字符串是否包含指定子串
```Go
fmt.Println(strings.Contains("hello", "el")) // true

fmt.Println(strings.Contains("hello", "x")) // false
```

- `Index(s, substr string) int`：查找子串第一次出现的索引，不存在返回 `-1`
```Go
fmt.Println(strings.Index("hello", "l")) // 2 
fmt.Println(strings.Index("hello", "x")) // -1
```

- `LastIndex(s, substr string) int`：查找子串最后一次出现的索引
```Go
fmt.Println(strings.LastIndex("hello", "l")) // 3
```

- `HasPrefix(s, prefix string) bool`：判断字符串是否以指定前缀开头
```Go
fmt.Println(strings.HasPrefix("golang", "go")) // true
```

- `HasSuffix(s, suffix string) bool`：判断字符串是否以指定后缀结尾
```Go
fmt.Println(strings.HasSuffix("golang", "ang")) // true
```

## 字符串分割与拼接
- `Split(s, sep string) []string`：按分隔符分割字符串，返回字符串切片
```Go
fmt.Println(strings.Split("a,b,c", ",")) // [a b c]
```

- `Join(a []string, sep string) string`：按字符串切片按分隔符拼接成一个字符串
```Go
s := []string{"go", "is", "good"} 
fmt.Println(strings.Join(s, " ")) // go is good
```

- `Fields(s string) []string`：按空白字符分割字符串（空格、换行、制表符等）
```Go
fmt.Println(strings.Fields("go lang\nis good")) // [go lang is good]
```

## 字符串替换
- `Replace(s, old, new string, n int) string`：替换子串，`n`为替换次数（`-1`表示替换所有）
```Go
// 替换所有 
fmt.Println(strings.Replace("hello hello", "l", "x", -1)) // hexxo hexxo 
// 只替换1次 
fmt.Println(strings.Replace("hello hello", "l", "x", 1)) // hexlo hello
```

- `ReplaceAll(s, old, new string) string`：替换所有匹配的子串
```Go
fmt.Println(strings.ReplaceAll("hello hello", "l", "x")) // hexxo hexxo
```

## 大小写转换
- `ToLower(s string) string`：转小写
```Go
fmt.Println(strings.ToLower("GOLANG")) // golang
```

- `ToUpper(s string) string`：转大写
```Go
fmt.Println(strings.ToUpper("golang")) // GOLANG
```

## 字符串修剪（去除首尾字符）
- `TrimSpace(s string) string`：去除首尾空白字符
```Go
fmt.Println(strings.TrimSpace(" hello go ")) // hello go
```

- `Trim(s, sutset string) string`：去除首尾指定字符
```Go
// 去除首尾的 ! 和 空格 
fmt.Println(strings.Trim("!!!hello go!! ", "! ")) // hello go
```

- `TrimLeft/TrimRight`：只去除左侧/右侧指定字符
```Go
fmt.Println(strings.TrimLeft("!!!hello", "!")) // hello 
fmt.Println(strings.TrimRight("hello!!!", "!")) // hello
```

## 字符串比较与长度
- `Compare(a, b string) int`：比较字符串字典序（返回0表示相等）
```Go
fmt.Println(strings.Compare("a", "b")) // -1 
fmt.Println(strings.Compare("b", "a")) // 1 
fmt.Println(strings.Compare("a", "a")) // 0
```

- `EqualFold(a, b string) bool`：不区分大小写比较字符串是否相等
```Go
fmt.Println(strings.EqualFold("Go", "gO")) // true
```

- 字符串长度
	- 英文/数字：直接使用`len()`
	- 中文/Unicode：用`utf8.RuneCountInString()`
```Go
import "unicode/utf8" 

fmt.Println(len("hello")) // 5 
fmt.Println(utf8.RuneCountInString("你好")) // 2
```

## 其他高频函数
- `Count(s, substr string) int`：统计子串出现次数
```Go
fmt.Println(strings.Count("hello", "l")) // 2
```

- `Repeat(s string, count int) string`：重复字符串
```Go
fmt.Println(strings.Repeat("go", 3)) // gogogo
```

- `Map(mapping func(rune)rune, s string) string`：按自定义规则替换字符
```Go
// 把所有 l 替换成 x 
fmt.Println(strings.Map(func(r rune) rune { 
	if r == 'l' { 
		return 'x' 
	} 
	return r 
}, "hello")) // hexxo
```

## 切片基础操作
- `Equal`：判断两个切片是否完全相等：元素、顺序、长度都相同才返回true
```Go
a := []int{1,2,3} 
b := []int{1,2,3}

fmt.Println(slices.Equal(a, c)) // false
```

- `Clone`：深拷贝切片：创建独立副本，修改新切片不影响原切片
```Go

new[0] = 999 fmt.Println(old) // [1 2 3] 原切片不变
```

- `Compare`：字典序比较切片：返回：-1（a < b）、0（相等）、1（a > b）
```Go
a := []int{1,2} 
b := []int{1,3} 
fmt.Println(slices.Compare(a, b)) // -1
```

## 查找元素

```Go
s := []string{"go", "java", "python"}
```

- `Index`：查找元素第一次出现的索引：不存在返回-1

- `LastIndex`：查找元素最后一次出现的索引

## 结构体排序
这是`sort`包最核心用途

### 实现`sort.Interface`（传统方法）
==**必须实现3个方法：`Len`、`Less`、`Swap`**==
```Go
type User struct { 
	Name string 
	Age int 
} 

// 定义切片类型 
type UserSlice []User 

// 实现3个方法 
func (u UserSlice) Len() int { 
	return len(u) 
} 

func (u UserSlice) Swap(i, j int) { 
	u[i], u[j] = u[j], u[i] 
} 

func (u UserSlice) Less(i, j int) bool { 
	return u[i].Age < u[j].Age 
} 

// 使用 
users := []User{{"A",20}, {"B",18}} 
sort.Sort(UserSlice(users)) // 按年龄升序
```

### `sort.Slice`更简单（推荐）
不用实现接口，直接传匿名函数，开发最常用
```Go
users := []User{{"A",20}, {"B",18}} 

// 按年龄升序 
sort.Slice(users, func(i, j int) bool { 
	return users[i].Age < users[j].Age 
})
```

**降序**：
```Go
return users[i].Age > users[j].Age
```

**多字段排序**：
```Go
// 先按年龄，年龄相同按名字 
return users[i].Age < users[j].Age || 
(users[i].Age == users[j].Age && users[i].Name < users[j].Name)
```

```Go
line, _ := rd.ReadBytes('\n')
```

# strconv
导入：
```Go
import "strconv"
```

## 字符串→整数
- `strconv.Atoi(s)`：
	- A = ASCII
	- to = 转成
	- i = int
```Go
s := "123" 
num, _ := strconv.Atoi(s) // num = 123
```

## 整数→字符串
- `strconv.Itoa`：
	- I = int
	- to = 转成
	- a = ASCII
```Go
num := 123 
s := strconv.Itoa(num) // s = "123"
```

## 字符串→布尔值
```Go
strconv.ParseBool("true") // true 
strconv.ParseBool("false") // false
```

## 字符串→浮点数
```Go
f, _ := strconv.ParseFloat("3.14", 64)
```
## 浮点数→字符串
```Go
s := strconv.FormatFloat(3.14, 'f', 2, 64) // "3.14"
```

## 字符串→指定位数的整数
```Go
num, _ := strconv.ParseInt("123", 10, 64) 
// 10 = 十进制 
// 64 = int64
```
