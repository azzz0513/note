## RPC
**RPC**（Remote Procedure Call，远程过程调用）：是一种通信模型或框架，目标是远程服务调用像调用本地函数一样简单。它关注的是方法或过程的调用，隐藏了底层的网络细节。

RPC通常是面向动作的，直接调用远程服务的方法（如`userService.GetUser(123)`）

适合用于微服务内部通信（如服务之间高频调用），需要高性能和强类型约束的场景。

## gprc
**grpc**是一个高性能、开源和通用的RPC框架，面向移动和HTTP/2设计。

## 原生rpc

```Go
// server.go
package main  
  
import (  
    "fmt"  
    "net"    
    "net/http"    
    "net/rpc"
)  
  
type Server struct {  
}  
  
type Req struct {  
    Num1 int  
    Num2 int  
}  
  
type Res struct {  
    Num int  
}  
  
func (s Server) Add(req Req, res *Res) error {  
    fmt.Printf("请求来了: {%d, %d}\n", req.Num1, req.Num2)  
    res.Num = req.Num1 + req.Num2  
    return nil  
}  
  
func main() {  
	// 注册rpc服务  
	rpc.Register(new(Server))  
	// 设置HTTP服务器， 将RPC服务映射到HTTP端点，使客户端可以通过HTTP协议进行RPC调用  
	rpc.HandleHTTP()  
	// 创建网络监听器  
	listen, err := net.Listen("tcp", ":8081")  
	if err != nil {  
	    fmt.Println(err)  
	    return  
	}  
	// 启动HTTP服务器  
	http.Serve(listen, nil)  
}
```

```Go
// client.go
package main  
  
import (  
    "fmt"  
    "net/rpc"
)  
  
type Req struct {  
    Num1 int  
    Num2 int  
}  
  
type Res struct {  
    Num int  
}  
  
func main() {  
	// 创建请求参数  
	req := Req{2, 2}  
	// 建立RPC客户端连接，通过HTTP协议连接到服务器的8081端口，返回一个RPC客户端实例用于后续调用  
	client, err := rpc.DialHTTP("tcp", ":8081")  
	if err != nil {  
	    fmt.Println(err)  
	    return  
	}  
	// 响应结构体  
	var res Res  
	// 执行RPC调用  
	client.Call("Server.Add", req, &res)  
	fmt.Println(res)  
}
```
不足：
- 编写相对复杂，需要自己关注实现过程
- 没有代码提示，容易写错

golang写RPC程序，必须符合4个基本条件：
- 结构体字段首字母要大写，可以被其他文件调用
- 函数名必须首字母大写
- 函数第一参数是接收参数，第二个参数是返回给客户端的参数，必须是指针类型
- 函数还必须有一个返回值error

### RPC调用流程
- 微服务架构下，数据交互一般是对内RPC，对外REST
- 将业务按功能模块拆分到各个微服务，具有提高项目协作效率、降低模块耦合度、提高系统可用性等优点，但是开发门槛比较高，比如 RPC 框架的使用、后期的服务监控等工作
- 一般情况下，我们会将功能代码在本地直接调用，微服务架构下，我们需要将这个函数作为单独的服务运行，客户端通过网络调用

```shell
protoc -I . --go_out=plugins=grpc:.\grpc_proto .\grpc_proto\hello.proto
```
- `protoc`：用于将`.proto`文件编译成目标语言的代码
- `-I .`：`-I`指定import路径，`.`表示当前目录作为proto文件的搜索路径
	- 当你的`.proto`文件中有`import`语句时，`protoc`会在`-I`指定的路径下查找被导入的文件。
- `--go_out=plugins=grpc:.\grpc_proto`：
	- `--go_out`：指定生成Go语言代码
	- `plugins=grpc`：启用grpc插件，生成gRPC服务相关代码
	- `:.\grpc_proto`：指定输出目录
- `.\grpc_proto\hello.proto`：指定要编译的proto文件路径

### 服务端
1. 编写一个结构体
2. 重要的是得实现protobuf中的所有方法
3. 监听端口
4. 注册服务
```Go
package main  
  
import (  
    "context"  
    "fmt"    
    "google.golang.org/grpc"    
    "google.golang.org/grpc/grpclog"    
    "grpc_study/grpc_proto/hello_grpc"    
    "net")  
  
type HelloService1 struct{}  
  
func (HelloService1) SayHello(ctx context.Context, request *hello_grpc.HelloRequest) (res *hello_grpc.HelloResponse, err error) {  
    return &hello_grpc.HelloResponse{  
       Name:    "张三",  
       Message: "ok",  
    }, nil  
}  
  
func main() {  
    // 监听端口  
    listen, err := net.Listen("tcp", ":8081")  
    if err != nil {  
       grpclog.Fatalf("failed to listen: %v", err)  
    }  
  
    // 创建一个grpc服务器实例  
    s := grpc.NewServer()  
    server := HelloService1{}  
    // 将server结构体注册为grpc服务  
    // 将HelloService1实例注册到grpc服务器中，使得客户端可以使用SayHello方法  
    hello_grpc.RegisterHelloServiceServer(s, &server)  
    fmt.Println("grpc server start at :8081")  
    // 开始处理客户端请求  
    // 这是一个阻塞调用，服务器持续运行到程序终止  
    err = s.Serve(listen)  
}
```

### 客户端
1. 建立连接
2. 调用方法
```Go
package main  
  
import (  
    "context"  
    "fmt"    
    "google.golang.org/grpc"    
	"google.golang.org/grpc/credentials/insecure"   
	"grpc_study/grpc_proto/hello_grpc"    
    "log")  
  
func main() {  
    // 定义要连接的grpc服务器地址和端口号  
    addr := ":8081"  
    // 使用grpc.Dial创建一个到指定地址的grpc连接  
    conn, err := grpc.Dial(addr, grpc.WithTransportCredentials(insecure.NewCredentials()))  
    if err != nil {  
       log.Fatalf("gprc connect addr %s failed, err: %v", addr, err)  
    }  
    defer conn.Close()  
  
    // 初始化客户端  
  
    client := hello_grpc.NewHelloServiceClient(conn)  
    // 调用服务器端的SayHello方法  
    // context.Background()提供默认的上下文环境  
    // 传入HelloRequest结构体作为参数  
    result, err := client.SayHello(context.Background(), &hello_grpc.HelloRequest{  
       Name:    "张三",  
       Message: "ok",  
    })  
    fmt.Println(result, err)  
}
```

```Go

	}
```

```Go
type Q1 struct {
```

#### 编写风格
1. 文件名建议下划线
2. 包名和目录名对应
3. 服务名、方法名、消息名均为大驼峰
4. 字段名为下划线

## 多服务
```Go
syntax = "proto3";  
option go_package = "/multi_grpc";  
  
service VideoService {  
    rpc Look (Request) returns (Response) {}  
}  
  
message Request {  
    string name = 1;  
}  
  
message Response {  
    string name = 1;  
}  
  
service OrderService {  
    rpc Buy (Request) returns (Response) {}  
}
```

服务端：
```Go
package main  
  
import (  
    "context"  
    "fmt"    
    "google.golang.org/grpc"    
    "grpc_study/grpc_proto/multi_grpc"    
    "net")  
  
type videoService struct{}  
  
func (videoService) Look(ctx context.Context, request *multi_grpc.Request) (*multi_grpc.Response, error) {  
    fmt.Println("video: ", request.Name)  
    return &multi_grpc.Response{  
       Name: "张三",  
    }, nil  
}  
  
type orderService struct{}  
  
func (orderService) Buy(ctx context.Context, request *multi_grpc.Request) (*multi_grpc.Response, error) {  
    fmt.Println("order: ", request.Name)  
    return &multi_grpc.Response{  
       Name: "张三",  
    }, nil  
}  
  
func main() {  
    // 监听端口  
    listen, err := net.Listen("tcp", ":8081")  
    if err != nil {  
       fmt.Println(err)  
       return  
    }  
    // 创建一个grpc服务器实例  
    s := grpc.NewServer()  
    // 将server结构体注册为grpc服务  
    multi_grpc.RegisterVideoServiceServer(s, &videoService{})  
    multi_grpc.RegisterOrderServiceServer(s, &orderService{})  
    fmt.Println("grpc server start at :8081")  
    err = s.Serve(listen)  
}
```

客户端：
```Go
package main  
  
import (  
    "context"  
    "google.golang.org/grpc"    
    "google.golang.org/grpc/credentials/insecure"    
    "grpc_study/grpc_proto/multi_grpc")  
  
func main() {  
    addr := ":8081"  
  
    // 建立与gRPC服务器的连接，使用不安全的传输凭据  
    conn, err := grpc.Dial(addr, grpc.WithTransportCredentials(insecure.NewCredentials()))  
    if err != nil {  
       panic(err)  
    }  
    defer conn.Close()  
  
    // 创建多个服务的客户端实例  
    videoClient := multi_grpc.NewVideoServiceClient(conn)  
    orderClient := multi_grpc.NewOrderServiceClient(conn)  
      
    res1, err := videoClient.Look(context.Background(), &multi_grpc.Request{Name: "电影"})  
    if err != nil {  
       panic(err)  
    }  
    println("video res: ", res1.Name)  
  
    res2, err := orderClient.Buy(context.Background(), &multi_grpc.Request{Name: "商品"})  
    if err != nil {  
       panic(err)  
    }  
    println("order res: ", res2.Name)  
}
```
